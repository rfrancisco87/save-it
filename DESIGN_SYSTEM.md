# Design System - Save It

This document outlines the visual style for the "Save It" Chrome extension, inspired by Notion's minimalist aesthetic.

## Typography
*   **Font Family**: Inter, -apple-system, BlinkMacSystemFont, "Segoe UI", Helvetica, "Apple Color Emoji", Arial, sans-serif, "Segoe UI Emoji", "Segoe UI Symbol".
*   **Base Size**: 14px.
*   **Line Height**: 1.5.

## Colors

### Core
*   **Background**: `#FFFFFF` (White)
*   **Text Primary**: `#37352F` (Notion Black)
*   **Text Secondary**: `#9B9A97` (Notion Grey)
*   **Border**: `#E0E0E0`

### Interactions
*   **Hover Background**: `#F1F1EF`
*   **Selection**: `#D9EFF8`

### Brand/Actions
*   **Primary Action (Blue)**: `#2EAADC`
*   **Destructive/Error (Red)**: `#EB5757`
*   **Success (Green)**: `#448361`

## Components

### Buttons
*   **Structure**: 4px border radius, 0 1px 2px rgba(15, 15, 15, 0.1).
*   **Standard**: White background, border, black text.
*   **Primary**: Blue background, white text, no border.
*   **States**: 
    *   Hover: Slight opacity change or background darken.
    *   Active: Scale down slightly (0.98).

### Inputs / TextAreas
*   **Structure**: No visible border by default (seamless) or subtle light grey border.
*   **Focus**: Light blue ring or darker border.
*   **Padding**: 8px 12px.

### Layout
*   **Spacing**: 8px base unit.
*   **Container**: 300px - 400px width for popup.
